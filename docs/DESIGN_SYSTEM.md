# Harmony Design System

## Design Philosophy
- Clarity before decoration
- Speed before animation
- Consistency before customization

## Visual Identity
- Premium but approachable
- Modern without appearing experimental
- Futuristic without feeling overwhelming

## Core Principles
- Minimal cognitive load
- Information density without clutter
- Keyboard-first workflows where possible
- Consistent interaction patterns

## Theme Support
- Light
- Dark
- System

## Initial Typography Direction
- Clean sans-serif typography
- Strong readability
- Comfortable spacing for long sessions

## Motion Principles
- Purposeful animations only
- Fast transitions
- No distracting effects
